# ZEPTO-DASHBOARD
Project Title / Headline 
ZEPTO SALES ANALYSE: Zepto Sales & Operations Report And Dashboard

Zepto Sales & Operations Analytics Dashboard is a visually engaging and analytical Power BI report designed to help stakeholders to track year to year and month to month growth of the company this dashboard also report which product category  is catching more customers so that we know the companies strength and weakness.
This dashboard also disclose highest contributing region and country towards sales and profit.

Tech stack 
Power BI Desktop - Main data visualisation platform used for report creation.
Power Query - Data transformation and cleaning layer for reshaping and preparing  the data.
Dax (Data Analysis Expressions) - Used for calculated measures, dynamic visuals, and conditional logic.
Data Modeling - Relationships established among tables to enables cross-filtering and aggregation.
File Format - .pbix for development  and .png for dashboard previews

Data source: kaggle.com
